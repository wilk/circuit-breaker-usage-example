# Circuit Breaker usage example

## Install

Run nvm to setup the correct version of Node.js

```bash
$ nvm use
```

Install the dependencies via NPM:

```bash
$ npm i
```

## Running

Run GestioneFatture:

```bash
$ npm run start:gestione-fatture
```

Run ControlloFatture:

```bash
$ npm run start:controllo-fatture
```
